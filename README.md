# 🌸 Harém Manager

O **Harém Manager** é uma aplicação desktop desenvolvida com Electron para a gestão de personagens em campanhas de RPG. O sistema organiza as informações em cards visuais, permitindo a visualização rápida de atributos e imagens das personagens.

---

## 🚀 Tecnologias Utilizadas

Este projeto foi construído utilizando o ecossistema Web adaptado para Desktop:

* **Linguagem:** HTML5, CSS3 e JavaScript (ES6+).
* **Framework Desktop:** [Electron](https://www.electronjs.org/) (para execução como programa nativo).
* **Estilização:** CSS moderno com foco em layouts responsivos (Cards e Sidebars).

---

## ✨ Funcionalidades

O sistema foca na organização visual e facilidade de acesso aos dados dos jogadores:

* **Cadastro de Perfil:** Gerenciamento de informações dos jogadores da campanha.
* **Gestão de Waifus:** Cadastro detalhado de personagens, incluindo:
    * Upload e exibição de imagens.
    * Cards informativos com atributos de RPG.
    * Organização em grade (Grid System).
* **Interface Desktop:** Experiência de software nativo com navegação fluida.

## 🛠️ Como Executar o Projeto

Para rodar o Harém Manager localmente, você precisará do [Node.js](https://nodejs.org/) instalado.

1.  **Clone o repositório:**
    ```bash
    git clone [https://github.com/seu-usuario/harem-manager.git](https://github.com/seu-usuario/harem-manager.git)
    ```

2.  **Entre na pasta do projeto:**
    ```bash
    cd harem-manager
    ```

3.  **Instale as dependências:**
    ```bash
    npm install
    ```

4.  **Inicie a aplicação:**
    ```bash
    npm start
    ```
